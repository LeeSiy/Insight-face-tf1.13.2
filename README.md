## Version Info
  - 1. Training insight face
    * Python 3.6.9
    * mxnet 1.5.0
    * Pytorch 1.6.0
    * Torch Vision 0.7.0
    * Cuda 10.1

  - 2. Training retina face and Convert model
    * Python 3.6.9
    * Tensorflow 1.13.2
    * Cuda 10.0
